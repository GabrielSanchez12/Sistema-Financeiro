
# Sistema Financeiro

## Descrição
Sistema financeiro simples desenvolvido em Python utilizando Programação Orientada a Objetos (POO).

Dentro do sistema, o usuário consegue:
- Adicionar receitas
- Adicionar despesas
- Visualizar saldo
- Calcular juros simples
- Calcular juros compostos
- Visualizar histórico de movimentações
- Remover movimentações
- Calcular juros simples
- Calcular juros compostos

O sistema possui interface gráfica (GUI) desenvolvida com Tkinter e persitência de dados em formato JSON, permitindo que as informações sejam mantidas mesmo após o fechamento do programa.

---

## Execução

1. Abra o projeto em uma IDE Python (PyCharm, VSCode, etc.)
2. Execute o arquivo `main.py`
3. Utilize a interface gráfica para gerenciar as movimentações financeiras!

---

## Tecnologias Utilizadas
- Python
- Tkinter
- JSON
- Programação Orientada a Objetos (POO)
